## 🕰️ 개발 기간


### ⚙️ 개발 환경
- **Framework** : React
- vite (Vite는 go로작성된 esbuilder를 사용해서 Webpack5보다 최대 100배 빠른 빌드속도를 가짐)
- react-router-dom : 6.14.2
- styled-components : 6.0.7
- firebase : 10.1.0
